# SLAI Company Engine

**Status:** Future planning only  
**Active build priority:** ServicesOS remains priority one until beta-critical workflows are stable.  
**Purpose:** Define how SLAI OS, GrowthAI, Research Agent, Client Finder, and ServicesOS operating data work together to help Stellar Logic AI operate and grow.

---

## Core Framing

```text
SLAI Company Engine =
SLAI OS + GrowthAI + Research Agent + Client Finder + ServicesOS operating data
```

Simple version:

```text
ServicesOS creates operating data.
GrowthAI turns that data into revenue opportunities.
Research Agent collects outside intelligence.
Client Finder identifies good-fit customers.
SLAI OS shows Jamie what matters and what action to take.
```

This is not an active product build yet. It is a blueprint for how SLAI can operate internally and later support customers with growth intelligence.
