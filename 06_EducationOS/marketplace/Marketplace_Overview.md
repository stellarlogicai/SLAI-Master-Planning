# EducationOS Marketplace Overview

Educators can publish, sell, and share learning content. A teacher can create a lesson during class, save it, edit it, and publish it.

## Content
Lesson plans, quizzes, worksheets, study guides, flash cards, homework, slides, interactive lessons, practice exams, AI simulations, recorded courses, curriculums, workshops, certification courses.

## Philosophy
Teachers already create valuable content. EducationOS helps turn that work into reusable income while taking a fair marketplace fee.
