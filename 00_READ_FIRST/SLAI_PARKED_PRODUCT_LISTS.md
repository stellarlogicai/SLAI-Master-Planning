# Promotion Rules

A parked list can become live only if it passes at least one of these tests:

1. It directly helps ServicesOS reach beta stability.
2. It directly helps ServicesOS get customers.
3. It directly reduces founder workload without creating new product complexity.
4. It directly supports revenue.
5. ServicesOS is already stable enough that the next product can safely begin.

If it does not pass one of those tests, it stays parked.
