# EducationOS and ComplianceAI Relationship

## Core Difference

EducationOS teaches people how to do something.

ComplianceAI makes sure required things actually happen.

```text
EducationOS = knowledge layer
ComplianceAI = accountability layer
SLAIOS = memory/proof layer
```

## Example: Chemical Safety

```text
EducationOS
↓
Employee completes chemical safety training
↓
Quiz is passed
↓
Certificate is issued
```

Then later:

```text
ComplianceAI
↓
Certificate renewal date approaches
↓
Employee/manager gets task
↓
Renewal training is assigned
↓
Completion is verified
↓
Audit trail is saved
```

## Example: Pharmacy Outdates

```text
EducationOS
↓
Teaches employee how outdates, pull windows, RGM, toxic totes, and scan-out rules work
```

Then:

```text
ComplianceAI
↓
Generates outdate task
↓
Applies department pull rule
↓
Guides scan-out and disposal routing
↓
Requires manager/pharmacist verification if needed
↓
Stores audit trail
```

## Full Loop

```text
Teach
↓
Execute
↓
Verify
↓
Remember
↓
Improve
```
