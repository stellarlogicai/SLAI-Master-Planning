# GrowthAI Memory System

## Purpose

GrowthAI memory improves outreach quality over time.

## Memory Types

### Working Memory

Current lead, current draft, current conversation.

### Short-Term Memory

Recent interactions, follow-ups, objections, active campaigns.

### Long-Term Memory

Persistent preferences and patterns.

Examples:

- Customer prefers email over phone.
- Cleaning companies object to price first.
- Businesses with 5-15 employees convert best.
- Founder-led messages perform better early.

### Institutional Memory

Human-approved lessons used across SLAI.

Examples:

- ServicesOS best customer profile
- Winning outreach angles
- Ethical outreach rules
- Industry playbooks

## Aging Rule

Long-term memory should have limits.

Older memories should be summarized and archived, not deleted.

## Human Review

New learning items should be backlogged for weekly review.

Humans approve, reject, or modify important institutional memories.
