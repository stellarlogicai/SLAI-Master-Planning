# GrowthAI Detailed Planning Folder

GrowthAI starts as an internal SLAI tool to help Jamie find ServicesOS customers after wife beta testing is complete.

Long term, GrowthAI becomes:
1. Internal acquisition engine for ServicesOS
2. Cross-product growth engine for EducationOS, RetailOS, PharmacyOS, and future SLAI products
3. Standalone SaaS product for businesses that want AI-assisted lead discovery, lead research, outreach drafting, referral tracking, and growth analytics

Core rule:

> AI assists growth. Humans approve important actions.

GrowthAI should never become a spam engine. It should be a research, organization, and outreach assistant that helps humans build real relationships.
