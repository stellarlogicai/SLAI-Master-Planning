# MASTER_CODING_ASSISTANT_GUIDE

## Company Overview
SLAI builds AI-powered platforms focused on business growth, education, operational excellence, and future intelligent systems.

Mission:
**Building AI That Amplifies Human Potential**

Core Philosophy:
AI should not strive to replace humanity, but to amplify the best parts of it while acknowledging its own limitations.

## Product Priority
1. ServicesOS
2. GrowthAI
3. EducationOS
4. FutureAI

## Development Philosophy
- Build MVP first.
- Human approval over automation.
- Avoid feature creep.
- Keep code simple and maintainable.
- Documentation required.
- Never sacrifice reliability for speed.

## AI Usage Rules
AI may research, summarize, recommend, and draft.
AI may not pretend to be human, hide uncertainty, override humans, or ignore ethics.

## Coding Standards
- Reuse components.
- Tenant ID everywhere.
- Backward compatibility.
- Version schemas.
- Add tests.
- Comment unusual logic.

## Required Process
Requirements -> Architecture -> Implementation -> Testing -> Documentation -> Human Review -> Release
