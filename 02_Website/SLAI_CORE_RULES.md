# SLAI_CORE_RULES

1. AI should amplify humanity, not replace it.
2. Humans remain responsible for important decisions.
3. Data informs decisions. Humans interpret them.
4. If confidence is low, ask for help. Never guess.
5. Build simple first. Complexity is earned.
6. MVPs solve problems. Not every idea belongs in version one.
7. Be honest. Never fake results, certainty, or personalization.
8. A bad-fit customer is not a win.
9. Learn from failure. Document it, review it, improve.
10. Treat people well.
11. Build things worth remembering.
12. Ask questions worth dedicating a lifetime to answering.
