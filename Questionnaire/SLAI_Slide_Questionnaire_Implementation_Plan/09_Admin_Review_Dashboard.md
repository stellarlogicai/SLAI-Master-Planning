# Admin Review Dashboard

## Purpose

The admin dashboard turns raw questionnaire responses into usable product research.

## V1 Dashboard Features

- List all responses
- Filter by product area
- Filter by respondent role
- Filter by review status
- Open individual response
- Add tags
- Add internal notes
- Mark reviewed
- Mark as potential backlog candidate

## Response Statuses

- New
- In Review
- Reviewed
- Needs Follow-Up
- Converted to Insight
- Archived

## Useful Tags

- Labor Waste
- Screen Bouncing
- Training Pain
- Compliance Risk
- Payment Friction
- Scheduling Pain
- Communication Pain
- AI Opportunity
- Not MVP
- Future Feature

## Insight Workflow

```text
Raw response
↓
Human reads answer
↓
Human tags pain points
↓
Similar responses grouped
↓
Insight created
↓
Backlog candidate drafted
↓
Human approves or rejects
```

## AI Enhancement Later

AI can assist by:

- Summarizing long responses
- Detecting repeated themes
- Suggesting tags
- Estimating severity
- Drafting backlog candidates
- Flagging sensitive content

AI should not auto-approve insights.
