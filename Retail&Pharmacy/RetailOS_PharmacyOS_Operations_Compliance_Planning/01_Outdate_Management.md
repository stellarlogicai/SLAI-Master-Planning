# Outdate Management

## Pain Point

Outdates waste labor hours because employees manually check every product, read tiny dates, decode manufacturer codes, and handle interruptions.

A single department like First Aid can take hours when mixed with customers and employee questions.

## Desired Workflow

1. Scan or enter product once.
2. Store SKU, department, location, expiration date, and pull rule.
3. Calculate pull date automatically.
4. Show daily/weekly pull list.
5. Employee checks only items due or approaching action.
6. Employee scans item to confirm.
7. System tells employee what to do with it.
8. Manager verifies if required.

## Key Data Fields

- Product name
- SKU / UPC / barcode
- Department
- Shelf location
- Expiration date
- Calculated pull date
- Status
- Disposal route
- Employee assigned
- Manager verification
- Notes / photo proof

## Statuses

- Good
- Expiring Soon
- Pull Today
- Expired
- Removed
- Scanned Out
- Routed
- Verified
- Manager Review Required
